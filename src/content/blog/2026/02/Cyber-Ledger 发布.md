:::note{type="success"}
历时几天的折腾，终于把我一直心心念念的记账 App —— `Cyber-Ledger` (赛博账本) 给敲出来了。没有花哨的云同步强制绑定，也没有各种冗杂的社区和广告，这只是一款单纯查账、记账的本地应用。
:::

### 缘起

其实一开始也没想搞个这么个东西，只是这学期经历的事情比较多，各种杂项开销也变得琐碎起来。加上我经常因为搞人工智能要租用 AutoDL 的算力显卡，或者去调用一些大模型的 API，这就导致我的花销里总有一笔很难用传统记账软件（比如某记、某手记）去精确归类的“赛博账单”。

每到月底核算的时候，看着那些死板的分类和永远对不上的余额，我就开始后悔为什么没有早点用一种更符合极客直觉的方式把钱管起来。尤其是当你发现有一些开支明明是“待报销”的，但在那些传统的 App 里，它们也会无情地吃掉你当月的正常预算限额，这就让人很焦虑。

思来想去，干脆自己手搓一个吧。

### 它能做什么？

这并没有什么革命性的技术，用的也都是很常规的 React + Vite + Tailwind 栈，最后通过 Capacitor 套了个安卓的壳子。但我把所有让我觉得“痛”的地方都修补了：

1. **绝对掌控的本地数据**：
   我太反感现在的软件动不动就要把账本传到他们的服务器上了。这个 App 所有的账单数据都只存在本地手机里。如果实在想备份或者多端同步，我也做了一个基于 WebDAV 的私有云连接选项。没有乱七八糟的账号体系，只有开屏的一个 4 位 PIN 码锁防偷窥。

2. **开发者视角的精算器**：
   我专门在里面插了个功能，可以根据目前的汇率去模拟算力显卡（比如 4090 或 H800 的价格）和 LLM Token 的具体租用消耗。那些烧钱买算力的日子，现在终于能以精确的数字直观地显示在账本上了。

3. **防吃灰的“周期订阅”与结转**：
   我老是会忘了给某些小众服务器节点或者 Netflix 续订而导致服务停摆。现在内置了周期订阅监视器；不仅如此，这个月没吃完的餐饮费或者娱乐预算，下个月它会自动累加滚存过去（Rollover），不用再为了“清空预算”而刻意消费。

### 结语与体验

整个折腾的过程虽然遇到了不少排版上的麻烦，比如在小屏幕上由于字太多老是溢出，最后也被我硬生生把文案全砍成了两个字（比如“编辑”、“删除”）给治好了。

看着最终在终端跑完 `assembleDebug` 生成出 APK 安装包的那一刻，手机桌面上出现那个带有炫光紫背景的 `SYS.LEDGER` 图标，还是有一些成就感的。虽然比不上大佬们做的那些大工程，但这好歹是一笔一划解决掉自己真实痛点的东西。

目前这还是 `v1.0.0`，里面可能还有暗坑，但也算圆了自己一个造轮子的念想。如果你也和我一样，苦于找不到一个安静的、算力和传统开销能两手抓的本地账本，可以下载下来试试。代码是开源在我的 GitHub 上的，随缘做做更新吧。
